# WebSocket Chess

A browser-based real-time multiplayer chess application built with JavaScript and WebSockets. The project implements complete chess game logic, including move validation and synchronization between two players.

## Features

- Real-time multiplayer gameplay
- Complete chess rules implementation
- Move validation
- Check and checkmate detection
- Stalemate detection
- Castling
- Pawn promotion
- Game timers with increment
- Draw offers
- Resignation

## Technologies

- JavaScript (ES6)
- HTML
- CSS
- Node.js
- Socket.IO

## Requirements

- Node.js 18 or newer
- npm

## Installation

Clone the repository and install the dependencies:

```bash
git clone https://github.com/your-username/websocket-chess.git
cd websocket-chess
npm install
```

## Running the application

Start the server:

```bash
npm start
```

Then open your browser and navigate to:

```
http://localhost:3000
```

Open the application in two browser windows (or on two different devices connected to the same server) to start a multiplayer game.
